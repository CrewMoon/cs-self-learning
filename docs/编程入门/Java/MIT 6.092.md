# MIT 6.092: Introduction To Programming In Java

## 课程简介

- 所属大学：MIT
- 先修要求：无
- 编程语言：Java
- 课程难度：🌟🌟
- 预计学时：少于 20 小时

MIT的Java入门课程，包括了你快速入门Java所需的知识，从配环境到Java的基础概念讲解。不需要有任何编程基础。学完这门课想要进阶的可以学习 [MIT 6.005/6.031](../../软件工程/6031.md) 。
